# ITEC-65---MCCAT

## Project Description
  MCCAT is a web-based food ordering system developed as a course project for ITEC 65. The system allows users to browse food items, create an account, log in, and place orders through a simple and user-friendly interface. It aims to provide a convenient way for customers to explore menu offerings and submit orders online.
  
## Features

- User Registration
- User Login
- Browse Food Menu
- Place Orders
- Contact Form
- Responsive Website Design

## How to Run the Project

1. Download or clone the repository.
2. Import the `mccat_db.sql` file into MySQL using phpMyAdmin.
3. Open the project folder in Visual Studio Code.
4. Run the project using Live Server or any local web server.
5. Open `index.html` in your browser.

## Database

The project uses a MySQL database to store user information, orders, and other system data. The database file can be found in the `database` folder.
